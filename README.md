# Vagrantfiles

Samples of Vagrant files per OS (obsolete).

## Supported OSes:

* CentOS
  * 7.5
* Ubuntu
  * trusty64

## Prerequisites

* [Vagrant](https://www.vagrantup.com)
* [Virtualbox](https://www.virtualbox.org/wiki/Downloads)

## Workflow

```console
git clone https://github.com/theodore86/Vagrantfiles.git
cd Vagrantfiles/<distro>
vagrant up
```
